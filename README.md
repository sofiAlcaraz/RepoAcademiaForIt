# AcademiaForIt
¡Hola! Este es mi **repo padre**, que contiene dos submódulos: **backend** y **frontend**.  
Por motivos de **facilidad, claridad y escalabilidad**, he decidido mantener ambos proyectos en repos independientes y luego juntarlos en este repo padre.

### Cómo clono los repos?
- Podes clonarte los dos repos por separado, como normalmente lo haces.
- Podes clonar el repo padre con el siguiente comando: git clone --recurse-submodules <URL-del-repo-padre>.

### Espero esta info te sirva de ayuda! 
Si se te complica espero tu comentario y te ayudo a clonar y correr los proyectos.
